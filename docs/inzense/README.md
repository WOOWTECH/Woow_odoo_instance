## 2. 會員登入與註冊

### 2.1 登入頁面

前往 `https://inzense-odoo.woowtech.io/web/login` 進入登入頁面。

![登入頁面](images/ch02_01_login_page.png)

**操作步驟**：

1. **電子信箱**：輸入您的帳號（Email 格式）
2. **密碼**：輸入密碼
3. 點擊 **「登入」** 按鈕

### 2.2 登入後畫面

- **管理員/員工登入**：進入 Odoo 後台管理介面
- **會員登入**：進入會員中心 Portal 儀表板

![管理員登入後](images/ch02_02_admin_after_login.png)

### 2.3 註冊新會員

1. 在登入頁面點擊 **「沒有帳號嗎？」** 連結
2. 填寫姓名、Email、密碼
3. 點擊 **「註冊」**
4. 系統將發送驗證信至您的信箱

### 2.4 忘記密碼

1. 在登入頁面點擊 **「重設密碼」**
2. 輸入註冊時的 Email
3. 系統發送重設連結至信箱
4. 點擊連結設定新密碼

| 欄位 | 說明 |
|------|------|
| 計畫名稱 | 集點卡名稱（如：誠品板橋-品牌集點卡） |
| 點數 | 目前累積點數（以「點」為單位） |
| 代碼 | 集點卡識別碼 |
| 到期日 | 點數有效期限（永久或指定日期） |

### 4.2 如何累積點數

1. 於 POS 門市消費時，收銀人員選擇您的會員帳號
2. 結帳後系統自動依規則累積點數
3. 累積的點數可於下次消費時折抵使用

## 14. POS 銷售點 — 多門市管理與倉庫隔離

### 14.1 架構說明

```
中和研發製香廠（中央倉庫 / 生產基地）
 ├── 誠品板橋門市倉庫 ← 補貨路線
 ├── 雲彩軒倉庫 ← 補貨路線
 ├── 滿滿倉庫 ← 補貨路線
 ├── 黑橋牌戎館倉庫 ← 補貨路線
 ├── 名品花園倉庫 ← 補貨路線
 ├── 納百川倉庫 ← 補貨路線
 └── 壹品藥局倉庫 ← 補貨路線
```

### 14.2 庫存隔離原則

- 每間門市有**獨立的庫存**，互不影響
- POS 銷售時，**自動從該門市的倉庫扣除庫存**
- 門市之間**不可直接調撥**，必須透過中央倉庫中轉
- 各門市可在後台查看自己倉庫的即時庫存量

### 14.3 內部調撥（補貨）操作

**場景**：中和研發製香廠 → 誠品板橋門市補貨

1. 進入 **庫存** 模組 → **作業** → **調撥**
2. 點擊 **「新增」**
3. 設定：
   - **作業類型**：誠品板橋門市倉庫：內部調撥
   - **來源位置**：中和/庫存
   - **目的位置**：板橋誠品/庫存
4. 在「作業」頁籤加入要調撥的商品與數量
5. 點擊 **「確認」** → 系統建立調撥單
6. 實際出貨後點擊 **「驗證」** 完成調撥
7. 兩邊倉庫的庫存自動更新

## 18. 銷售訂單完整流程

### 18.1 流程概覽

```
建立報價單 → 確認訂單 → 出貨 → 開立發票 → 收款
  (draft)      (sale)    (done)   (posted)   (paid)
```

![銷售列表](images/ch18_01_sales_list.png)

### 18.2 步驟 1：建立報價單

1. 進入 **銷售** 模組 → 點擊 **「新增」**
2. 選擇客戶（從聯絡人中搜尋）
3. 在「訂單明細」頁籤加入商品：
   - 點擊「新增一行」
   - 選擇商品（可搜尋或掃描條碼）
   - 設定數量和單價
4. 確認金額後點擊 **「儲存」**（報價單狀態：草稿）

### 18.3 步驟 2：確認訂單

1. 報價單確認無誤後，點擊 **「確認」** 按鈕
2. 報價單自動轉為「銷售訂單」
3. 系統同時自動建立出貨單（送貨單）

### 18.4 步驟 3：出貨

1. 點擊訂單上方的 **「出貨」** 按鈕（或數字徽章）
2. 進入出貨單頁面
3. 確認出貨商品與數量
4. 點擊 **「驗證」** 完成出貨
5. 出貨單狀態變更為「完成」

### 18.5 步驟 4：開立發票

1. 回到銷售訂單頁面
2. 點擊 **「建立發票」** 按鈕
3. 選擇 **「一般發票」**（依已交付數量）
4. 發票自動建立為草稿狀態
5. 進入發票頁面，確認金額後點擊 **「過帳」**

### 18.6 步驟 5：收款

1. 在已過帳的發票頁面，點擊 **「登記付款」**
2. 選擇付款日記帳（現金/銀行）
3. 確認收款金額
4. 點擊 **「建立付款」**
5. 發票狀態自動變更為 **「已付款」**

## 29. 常見問題（FAQ）

### Q1：忘記密碼怎麼辦？
在登入頁面點擊「重設密碼」，輸入 Email 後收取重設連結。若未收到，請聯繫管理員。

### Q2：POS 結帳時找不到客戶？
確認客戶已在「聯絡人」模組中建立，且帳號狀態為啟用。可在 POS 搜尋欄輸入客戶名稱或電話搜尋。

### Q3：商品庫存數量不對？
進入 **庫存** → **產品** 選擇商品，查看「手上」數量。若有差異，使用「實際盤點」功能調整。

### Q4：如何查看特定門市的銷售報表？
進入 **銷售點** → **訂單** → 使用篩選器選擇特定門市（POS 配置）。

### Q5：優惠券無法使用？
請確認：(1) 優惠券未過期 (2) 已在 POS 中選擇客戶 (3) 訂單金額符合使用條件。

### Q6：如何從中央倉庫補貨到門市？
進入 **庫存** → **調撥** → 新增內部調撥，設定來源（中和/庫存）和目的（門市/庫存）。詳見 [第14章](#14-pos-銷售點--多門市管理與倉庫隔離)。

### Q7：發票如何作廢？
開啟發票 → 點擊「折讓單」建立沖銷發票。已過帳的發票不可直接刪除，只能透過折讓單沖銷。

### Q8：客戶要求月結怎麼處理？
POS 結帳時選擇「客戶帳戶」付款方式（需先指定客戶），月底再透過後台統一建立發票和收款。

### Q9：如何新增門市員工帳號？
**設定** → **使用者** → **新增** → 填寫姓名和 Email → 設定權限群組（POS/庫存/銷售）→ 儲存。

### Q10：網站上的聯絡表單送到哪裡？
送到 **CRM** 模組，自動建立線索（Opportunity）。在 CRM Pipeline 中查看和處理。詳見 [第28章](#28-crm-客戶關係管理)。

### Q11：如何在電商網站上架新商品？
在後台商品頁面點擊「前往網站」→ 設定圖片和描述 → 點擊「發布」。詳見 [第21章](#21-後台管理--產品目錄管理)。

### Q12：禮品卡和電子錢包有什麼不同？
禮品卡有到期日且可贈送給他人（有代碼）；電子錢包綁定個人帳號、永久有效，類似儲值帳戶。
